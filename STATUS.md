# Статус серверов

## Текущее состояние

### ✅ Фронтенд
- **Порт:** 3000
- **Статус:** Запускается
- **URL:** http://localhost:3000
- **PID:** 88492

### ⚠️ Бэкенд
- **Порт:** 9999
- **Статус:** Запускается в отдельном окне
- **URL:** http://localhost:9999

## Что делать дальше

1. **Дождитесь запуска обоих серверов** (обычно 10-15 секунд)

2. **Проверьте фронтенд:**
   - Откройте: http://localhost:3000
   - Должна загрузиться главная страница

3. **Проверьте бэкенд:**
   - Откройте: http://localhost:9999/api/dashboard/stats
   - Должен вернуться JSON

4. **Если есть ошибки:**
   - Проверьте окно с бэкендом - там будут логи
   - Проверьте консоль браузера (F12) для ошибок фронтенда

## Быстрый перезапуск

Если нужно перезапустить оба сервера:

**Фронтенд:**
```powershell
cd frontend
npm run dev
```

**Бэкенд:**
```powershell
cd E:\HttpServer
$env:ARLIAI_API_KEY='597dbe7e-16ca-4803-ab17-5fa084909f37'
go run -tags no_gui main_no_gui.go
```

Или используйте готовые скрипты:
- `start-all.ps1` - запускает оба сервера
- `start-backend-exe.bat` - только бэкенд
