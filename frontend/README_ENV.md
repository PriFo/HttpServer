# Переменные окружения

## Настройка переменных окружения

Создайте файл `.env.local` в директории `frontend/` со следующими переменными:

```env
# URL backend сервера
BACKEND_URL=http://localhost:9999

# Имя сервисной базы данных (отображается в интерфейсе)
SERVICE_DB_NAME=Сервисная БД
```

## Описание переменных

- `BACKEND_URL` - URL адрес backend сервера (по умолчанию: `http://localhost:9999`)
- `SERVICE_DB_NAME` - Имя сервисной базы данных, которое отображается в интерфейсе (по умолчанию: `Сервисная БД`)

## Пример использования

Если вы хотите изменить имя сервисной БД на "Системная БД", добавьте в `.env.local`:

```env
SERVICE_DB_NAME=Системная БД
```

После изменения переменных окружения перезапустите Next.js dev server.
