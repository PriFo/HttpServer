## Компоненты

### ProjectSelector
- Выбор клиента и проекта
- Автоматическая загрузка данных
- Кнопка обновления

### NormalizationProcessTab
- Поддержка работы с проектом
- Загрузка и отображение информации о проекте
- Валидация и обработка ошибок
- Автоматическое и ручное обновление

## API Endpoints

1. `GET /api/clients` - список клиентов
2. `GET /api/clients/{clientId}/projects` - список проектов клиента
3. `GET /api/clients/{clientId}/projects/{projectId}` - информация о проекте
4. `GET /api/clients/{clientId}/projects/{projectId}/databases?active_only=true` - список активных БД
5. `POST /api/clients/{clientId}/projects/{projectId}/normalization/start` - запуск нормализации для проекта

## Примеры использования

### Запуск для всех БД проекта:
1. Выберите клиента
2. Выберите проект
3. Нажмите "Запустить нормализацию для N БД проекта"

### Обновление информации:
- Кнопка обновления информации о проекте (слева от кнопки статуса)
- Кнопка обновления статуса нормализации (справа)
