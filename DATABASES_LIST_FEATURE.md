# Отображение списка баз данных проекта

## ✅ Реализованная функциональность

### 1. Загрузка списка баз данных
- ✅ Автоматическая загрузка списка активных БД проекта
- ✅ Сохранение информации о каждой БД (id, name, file_path)
- ✅ Обновление списка при обновлении информации о проекте

### 2. Отображение списка
- ✅ Кнопка "Показать/Скрыть список баз данных"
- ✅ Раскрывающийся список с информацией о каждой БД
- ✅ Отображение названия и пути к файлу
- ✅ Прокручиваемый список (максимум 60px высоты)
- ✅ Красивое оформление с иконками

### 3. Автоматическое обновление
- ✅ Обновление списка после завершения нормализации
- ✅ Обновление при ручном обновлении информации о проекте
- ✅ Сброс списка при смене проекта

## Примеры отображения

### Кнопка показа списка:
```
Проект "Название проекта" (5 баз данных): обработка всех активных баз данных
[▼ Показать список баз данных (5)]
```

### Раскрытый список:
```
Базы данных, которые будут обработаны:
```

## Технические детали

### Состояние компонента:
- `projectDatabases` - массив объектов с информацией о БД
- `showDatabasesList` - флаг отображения списка

### Структура данных БД:
```typescript
{
  id: number
  name: string
  file_path: string
}
```

### Обработка данных:
- Автоматическое извлечение имени из пути, если name отсутствует
- Отображение только имени файла в списке (без полного пути)
- Уникальные ключи для каждого элемента списка

## Готово к использованию

Функция полностью реализована и готова к использованию. Пользователи могут видеть, какие базы данных будут обработаны при запуске нормализации для проекта.
