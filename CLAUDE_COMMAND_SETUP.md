# Настройка команды Claude

Команда `claude` настроена и готова к использованию!

## Использование

В PowerShell выполните:

```powershell
claude help          # Показать справку
claude server        # Запустить HTTP сервер
claude build         # Собрать Go приложение
claude test          # Запустить тесты
claude status        # Проверить статус сервера
```

## Файлы

- `claude.ps1` - Основной скрипт с функциональностью
- `claude.bat` - Batch-файл для вызова из командной строки
- `setup-claude-alias.ps1` - Скрипт настройки (уже выполнен)

## Примечание

Если команда `claude` не работает после перезапуска PowerShell:

1. Перезагрузите профиль: `. $PROFILE`
2. Или используйте напрямую: `.\claude.ps1 [command]`
3. Или используйте batch-файл: `.\claude.bat [command]`

## Добавление в PATH (опционально)

Если хотите использовать `claude` из любой директории, добавьте путь к проекту в переменную PATH:

```powershell
$env:PATH += ";E:\HttpServer"
```

Или добавьте в системные переменные окружения через настройки Windows.
