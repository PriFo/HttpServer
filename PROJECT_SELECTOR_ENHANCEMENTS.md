## Примеры отображения

### С информацией о проекте:
- **Описание:** `Проект "Название проекта" (5 баз данных): обработка всех активных баз данных`
- **Кнопка:** `Запустить нормализацию для 5 БД проекта`

### Без активных БД:
- **Описание:** `Проект "Название проекта" (0 баз данных): обработка всех активных баз данных`
- **Кнопка:** Заблокирована с подсказкой "В проекте нет активных баз данных"
- **Ошибка:** `В проекте нет активных баз данных для обработки`

## Технические детали

### API запросы:
1. `GET /api/clients/{clientId}/projects/{projectId}` - получение информации о проекте
2. `GET /api/clients/{clientId}/projects/{projectId}/databases?active_only=true` - получение списка активных БД

### Состояние компонента:
- `projectName` - название проекта
- `projectDatabasesCount` - количество активных баз данных

### Валидация:
- Проверка выполняется перед отправкой запроса на запуск
- Кнопка блокируется, если `projectDatabasesCount === 0`
